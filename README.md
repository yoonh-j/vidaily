vidaily
=======

요약
----
일상을 동영상으로 공유하는 안드로이드 앱입니다.

사용 언어
--
- Kotlin

사용 기술
---------
- AWS(Amplify, Cognito, S3, API)
- FCM
- ExoPlayer
- Glide
- Jetpack Navigation
- DataBinding
- LiveData
